## Get started in three steps

<Steps>

### Step 1: Get your API key

1. Sign up or log in at your [dashboard](https://search.leads-siren.com).
2. Navigate to your account settings to create an API key.
3. Copy and store your key securely — it is shown only once.

### Step 2: Make your first request

Use your API key to test the connection:

```bash
curl -X GET https://search.leads-siren.com/enrichment-service/test \
  -H "x-api-key: YOUR_API_KEY"
```

You should receive:

```json
{
  "status": "It worked."
}
```

### Step 3: Search for companies

Now try a real search:

```bash
curl -X POST https://search.leads-siren.com/enrichment-service/search_companies \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "data": {
      "filters": {
        "departement": ["75"],
        "activite_principale": ["62.01Z"],
        "etat_administratif": "A"
      },
      "page": 1
    }
  }'
```

</Steps>
