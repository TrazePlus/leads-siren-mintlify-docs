## Multi-language Examples

Use `<CodeGroup>` to show the same example in multiple languages:

<CodeGroup>
```bash cURL
curl -X POST https://search.leads-siren.com/enrichment-service/search_companies \
  -H "x-api-key: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"data": {"filters": {"departement": "75"}, "page": 1}}'
```

```python Python
import requests

resp = requests.post(
    "https://search.leads-siren.com/enrichment-service/search_companies",
    headers={"x-api-key": "YOUR_API_KEY"},
    json={"data": {"filters": {"departement": "75"}, "page": 1}}
)
```

```javascript JavaScript
const resp = await fetch(
  "https://search.leads-siren.com/enrichment-service/search_companies",
  {
    method: "POST",
    headers: {
      "x-api-key": "YOUR_API_KEY",
      "Content-Type": "application/json"
    },
    body: JSON.stringify({data: {filters: {departement: "75"}, page: 1}})
  }
);
```
