## Titles

Best practice is to use a single `#` for the page title (set via frontmatter) and `##` or `###` for section headings.

```md
## Heading 2
### Heading 3
#### Heading 4
```

## Text

Standard markdown formatting:

```md
**Bold text**
*Italic text*
~~Strikethrough~~
[Link text](https://example.com)
`inline code`
```

## Lists

```md
- Unordered item 1
- Unordered item 2

1. Ordered item 1
2. Ordered item 2
```

## Tables

```md
| Column 1 | Column 2 |
|----------|----------|
| Cell 1   | Cell 2   |
```

## Code Blocks

Use triple backticks with a language identifier:

````md
```python
print("Hello, world!")
```
````
